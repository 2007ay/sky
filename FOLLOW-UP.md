## Prerequisite
- Nodejs v6+

## Client setup (root folder : client)

- `npm install` (once)

## Server setup (Root folder : server)

- `npm install` (once)
- `npm run prep-prod` 
- `node src/server.js`

## How to  run application
- http://localhost:4000
- `queryParams`  app-util.js  can be used to modifiy search
